# 🎨 Visual Page Builder - Quick Start

## ✨ What is This?

The Visual Page Builder is a **simple, visual web editor** for anyone to update website content - no coding skills needed.

Think of it like a simplified version of Wix, WordPress, or Webflow - all the power, none of the complexity.

## 🚀 How to Access

1. **Go to**: Admin Dashboard (`/Frontend-admin/admin-dash.html`)
2. **Click**: **Visual Page Builder** (in the left sidebar)
3. **That's it!** You're ready to edit

## 📖 The 5-Step Process

### Step 1: Pick a Page
```
On the LEFT, click any page:
• Home
• Shop  
• About
• Contact
• Services
```

### Step 2: Click a Block
```
In the CENTER, click any content block.
It highlights in orange.
```

### Step 3: Edit on the Right
```
The RIGHT panel shows editable options.
Change the text, images, etc.
```

### Step 4: Save
```
TOP RIGHT: Click "Save" button
(saves to database)
```

### Step 5: Publish
```
TOP RIGHT: Click "Publish" button
(makes changes live on website)
```

That's literally it! 🎉

## 🎯 What You Can Do

| Want to... | Steps |
|-----------|-------|
| Change hero text | Click hero → Edit HTML → Save → Publish |
| Add images | Click gallery → Paste image URLs → Save |
| Update FAQ | Click FAQ → Edit JSON → Save |
| Reorder blocks | Drag blocks in center → Auto-saved |
| Add new block | Click button on left → Fills in defaults → Save |
| Delete block | Click block → Click delete → Confirm |

## 🖼️ Block Types Explained

**Text Block**
- Just type/paste content
- Use HTML like `<h2>Title</h2>` and `<p>text</p>`
- Good for: Bio, descriptions, lists, etc.

**Hero Block**
- Big colorful banner at top
- Edit: Title, description, maybe a button
- Good for: Page headers, catchy intros

**Image Block**
- Single image with caption
- Provide image URL + alt text
- Good for: Product photos, featured images

**Gallery Block**
- Multiple images in a grid
- Paste image URLs as JSON
- Good for: Portfolios, photo collections

**FAQ Block**
- Questions with expandable answers
- Provide Q&A pairs as JSON
- Good for: Common questions, help section

**CTA Block** (Call to Action)
- Button or action section
- Edit: Text, link, styling
- Good for: "Contact us", "Buy now", etc.

## 💻 HTML Quick Guide

You can use HTML in text blocks. Common examples:

```html
<h1>Big Title</h1>
<h2>Medium Title</h2>
<p>Regular paragraph text</p>

<strong>Bold text</strong>
<em>Italic text</em>

<a href="https://example.com">Link text</a>

<ul>
  <li>Bullet 1</li>
  <li>Bullet 2</li>
</ul>
```

Just copy/paste and customize!

## 🖼️ Image URLs

Where to get image URLs:

**From Google Images:**
1. Right-click image
2. Click "Copy image link"
3. Paste into Image URL field

**From Google Drive:**
1. Right-click image
2. Click "Open image in new tab"
3. Copy image URL from address bar
4. Paste into field

**From Imgur/Pinterest/etc:**
1. Right-click image
2. Copy image link
3. Paste into field

## ⚡ Pro Tips

✅ **See changes instantly** - The center shows live preview
✅ **Drag to reorder** - Just drag blocks up and down
✅ **Always Save first** - Then Publish when ready
✅ **Test on mobile** - Resize browser to check mobile view
✅ **Copy/paste from anywhere** - Paste content from Word, other sites, etc.

## ❌ Common Gotchas

❌ **Don't click Publish directly** - Click Save first, then Publish
❌ **Don't edit raw JSON if unsure** - Use simple text when possible
❌ **Don't use broken image links** - Test images load in new tab
❌ **Don't forget to close HTML tags** - `<p>` must have closing `</p>`

## 🆘 Something Broke?

**Page won't load?**
→ Refresh browser (Ctrl+R)

**Block disappeared?**
→ Click Save button manually

**Image not showing?**
→ Check URL by opening in new tab

**HTML not working?**
→ Check for typos in tags (e.g., `</p>` not `</p >`)
